# Atman's Prompt & Skills Repository

This repository serves as a personal collection of custom prompts, skills, and agent configurations for use with AI coding assistants.

## Contents

- **skills/**: A collection of specialized agent skills and instructions.
- **docs/**: Documentation and guidelines.
- **AGENTS.md**: Configuration and definitions for custom agents.

## Usage

These resources are designed to enhance the capabilities of AI agents by providing specific context and procedural knowledge.
